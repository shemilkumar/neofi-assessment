## Technologies Used
* React.js
* Typescript
* TailwindCSS
* Axios
* vite

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Run the development server:


```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
